Temperature Time Series Analysis

Source: Kaggle
The dataset contains average daily temperatures of Chennai, Tamil Nadu collected from the year 1990 to 2022. It has five parameters: Time, Average temperature (tavg), Minimum temperature(tmin), Maximum temperature(tmax), and precipitation(prcp).

Data Preprocessing

The dataset is loaded, and the 'time' and 'tavg' columns are selected. The 'time' column is converted to a datetime format, and the index is set to the 'time' column. Missing values are dropped, and the data is resampled to a monthly frequency.

Exploratory Data Analysis

Several EDA techniques are applied to understand the data better.

  1. Augmented Dickey-Fuller (ADF) Test:

    The ADF test is performed to check the stationarity of the time series data. The original temperature data and the seasonally differenced data are tested.

    • ADF test p-value: The p-value associated with the ADF test is 0.039283685...
    • Number of lags: The number of lags used in the ADF test is 12...
  2. Autocorrelation Function (ACF) and Partial Autocorrelation Function (PACF) Plots:

    ACF and PACF plots are generated to analyze the correlation in the differenced data. These plots help determine the values for p, d, and q in the ARIMA model.

  3. ARIMA Model:

    An ARIMA (1, 1, 1) model is built using the 'tavg' data. The model summary is displayed, providing information about the coefficients, standard errors, p-values, and other relevant statistics. A line plot is created to visualize the actual 'tavg' values and the predicted values.

  4. Seasonal ARIMA (SARIMA) Model:

    A seasonal ARIMA (1, 1, 1) (1, 1, 1, 12) model is built using the 'tavg' data. The model summary is displayed. A line plot is created to visualize the actual 'tavg' values and the forecasted values.

    • Model Specification: The ARIMA model only includes the ARIMA components (p, d, q), while the SARIMA model includes additional seasonal components (P, D, Q, s).

Report Summary:

The analysis begins with data preprocessing, ensuring the data is in the correct format and handling missing values. EDA techniques provide insights into the temperature data, including yearly and monthly trends. Seasonal decomposition helps identify the trend, seasonality, and residuals. The ADF test is performed to check the stationarity of the data, and the differenced data is used to generate ACF and PACF plots. Based on the analysis, both ARIMA and SARIMA models are built. The ARIMA model captures the non-seasonal components, while the SARIMA model accounts for both non-seasonal and seasonal components. The models' performance is evaluated by comparing the actual and predicted/forecasted values.

Overall, the analysis provides valuable insights into the temperature data, captures the underlying patterns, and generates forecasts using the ARIMA and SARIMA models.
